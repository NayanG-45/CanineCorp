# 🦴 CanineCorp: The Neural Recruitment OS // v3.1.0

> **"Fed up with human developers? Tired of them asking for 'equity' and 'work-life balance'? Meet your new masters."**

CanineCorp is a high-fidelity, cyberpunk-aesthetic recruitment platform that leverages advanced biometric scanning and neural-link processing to recruit the most elite canine developers in the sector. No more burnout. No more Jira tickets. Just raw, unbridled barking.

![Cyberpunk HUD](https://raw.githubusercontent.com/lucide-react/lucide/main/icons/bot.svg)

## ⚡ Features

- **Cinematic Entry Sequence**: High-tech glassmorphism intro with neural-link initialization.
- **3D Card Stack (Tactical Deck)**: Gesture-based candidate selection. Swipe left to PASS ON MUTT, swipe right to identify a GOOD BOY.
- **Biometric Interview Modal**: Vision-augmented AI resume generation. Scans candidate photos to determine technical roles like `Lead Bark-end Developer` or `Full Sniff Engineer`.
- **Dynamic Backgrounds**: Animated mesh-gradient HUD with neural status decorators.
- **Amber/Indigo Aesthetic**: A premium, brutalist-cyberpunk hybrid design language.

## 🛠 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router, Turbopack)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **AI Engine**: [Groq Cloud](https://groq.com/) (Llama 3.2 Vision & Llama 3.3 Versatile)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Data Source**: [Dog CEO API](https://dog.ceo/dog-api/)

## 🚀 Local Implementation

### Prerequisites
- Node.js 18+ 
- A Groq API Key

### Setup
1. **Clone the repository**:
   ```bash
   git clone [your-repo-link]
   cd Frontend-Sprint
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env.local` file in the root directory and add your key:
   ```env
   NEXT_PUBLIC_GROQ_API_KEY=your_gsk_xxx_key_here
   ```

4. **Initialize Neural Link**:
   ```bash
   npm run dev
   ```

5. **Access the Terminal**:
   Open [http://localhost:3000](http://localhost:3000)
